    1. vmware ubuntu 16.04 共享文件夹重启失效

    在 ~/.bashrc 添加 “sudo vmhgfs-fuse .host:/ /mnt/hgfs
    ”

    2. 执行npm ，再不用sudo

    sudo chown -R $(whoami) $(npm config get prefix)/{lib/node_modules,bin,share}

https://askubuntu.com/questions/955434/sudo-node-command-not-found

3. 更改文件夹权限(/opt)免除sudo

chmod -R a+r * 将目前目录下的所有档案与子目录皆设为任何人可读取
