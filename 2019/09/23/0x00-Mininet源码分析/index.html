Mininet是个很不错的模拟（emulate）网络的工具，特别在模拟SDN环境的时候。
使用Mininet可以在一台物理机上快速搭建较大规模的SDN网络进行控制器或交换机方面的验证和测试。

API
doc

一、整体逻辑

整体上，Mininet作为一个基于Python的网络模拟工具，可分为两大部分：

  1. Python库

    提供对网络中元素进行抽象和实现

  2. 运行文件

    基于库来完成各种自定义的模拟过程

三、源码

1. 结构

（1）bin/：包含模拟运行文件mn

（2）custom/：用户配置文件，拓扑图等

（3）mininet/：核心代码所在

a. cli模块：用于节点间的交互

link.TCIntf类：限制带宽、延迟、分组丢失率、最大排队长度等
link.Link类：表示链路，连接两个虚拟以太网接口

b. net模块：创建拓扑图，配置namespace，ip，连接控制器

c. log模块：记录日志

d. node模块:表示网络中的各个基本元素，也是最重要的

e. topo模块：维护网络拓扑的相关信息

classTopo( object ): 拓扑基类

（4）util/：辅助文件，脚本等
